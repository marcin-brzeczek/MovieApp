# MovieApp
Simple movie viewer app using OMDB API

## Architecture
 - \[x] Clean Architecture with MVVM as presentation layer

## Libraries
- \[x] Dagger
- \[x] Retrofit
- \[x] RxJava
- \[x] Databinding
- \[x] Paging
- \[x] Navigation
- \[x] Mockk

Note: Before using enter `YOUR_API_KEY` to omdb.properties file.
